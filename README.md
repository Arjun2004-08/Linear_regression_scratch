# 📈 Linear Regression from Scratch (Python)

This project implements **Linear Regression from scratch** using pure Python — without using libraries like scikit-learn.

It demonstrates how machine learning actually works under the hood using:

* Gradient Descent
* Mean Squared Error (Loss Function)
* R² Score (Model Evaluation)

---

## 🚀 Features

*  Manual implementation of Linear Regression
*  Gradient Descent optimization
*  Loss calculation (MSE)
*  R² Score evaluation
*  Clear understanding of ML fundamentals

---

## 🧮 Mathematical Concepts

### 1. Hypothesis Function

[
y = mx + b
]

---

## ⚙️ How It Works

1. Initialize parameters `m` and `b`
2. Predict values using current model
3. Compute loss (MSE)
4. Calculate gradients
5. Update parameters using Gradient Descent
6. Repeat for multiple epochs
7. Evaluate using R² score

---

## 🧪 Example Output

```
Epoch 1 → Loss: 120.45
Epoch 100 → Loss: 25.32
Epoch 500 → Loss: 5.87
```

Final Model:

```
y = 2.34x + 1.12
R² Score: 0.92
```

## 🧠 Why This Project?

Most people directly use libraries like TensorFlow or PyTorch, but this project helps you:

* Understand how models **learn step-by-step**
* Build strong ML fundamentals
* Prepare for interviews
* Debug models more effectively

---

## 📌 Key Learnings

* Gradient Descent is an **iterative optimization process**
* Loss decreases gradually — not instantly
* Model improves by **following the slope (gradient)**
* R² measures how well the model explains the data
